# Terre Anadolu Şarküteri

Terre Anadolu Şarküteri için hızlı, erişilebilir ve tamamen statik tanıtım sitesi.

## Yerelde çalıştırma

Herhangi bir derleme adımı yoktur. Proje klasöründe basit bir HTTP sunucusu açın:

```bash
python3 -m http.server 8080
```

Ardından `http://localhost:8080` adresini açın.

## Yayınlama

Dosyalar GitHub Pages, Cloudflare Pages, Netlify veya benzeri herhangi bir statik barındırma hizmetinde doğrudan yayınlanabilir.

## İçerik

- `index.html`: Sayfa içeriği ve SEO metadatası
- `styles.css`: Görsel sistem ve responsive düzen
- `script.js`: Mobil menü ve kaydırma animasyonları
- `assets/`: Logo ve hero görseli
